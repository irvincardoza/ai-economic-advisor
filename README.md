> Update: Model server is current down. I am working on integrating a different hugging face model due to licence issues.

## 🚀 Project Overview

Economic Intelligence Agents is an AI-powered financial assistant that leverages a multi-agent system to generate tailored economic insights, investment strategies, and risk assessments. Designed for businesses and investors, the system integrates FinGPT with FastAPI and CrewAI to deliver actionable financial reports.

---

## 🧠 Key Features

- 📊 **Real-time Market Analysis**  
  Automatically fetches live stock market data and evaluates macroeconomic trends.

- 🧩 **Multi-Agent Workflow (CrewAI)**  
  Specialized agents collaborate to deliver comprehensive reports:  
  - Market Analyst  
  - Investment Strategist  
  - Risk Manager  
  - Report Generator

- 📝 **Automated PDF Report Generation**  
  Generates professional financial reports summarizing recommendations and risks.

- 🔄 **Orchestration with FastAPI + CrewAI**  
  Seamlessly runs on a FastAPI server, coordinating all agents and generating outputs via API.

---

## 🏗 Tech Stack

- **Languages**: Python  
- **Frameworks**: FastAPI, CrewAI  
- **AI Models**: FinGPT via Hugging Face API  
- **Libraries**: ReportLab, YahooFinance, Requests, Pyngrok, Pydantic
